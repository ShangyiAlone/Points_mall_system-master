# Points_mall_system
积分商城系统 <br>
积分交易：用户可以通过完成特定的行为（例如购买商品、参与活动等）来获得积分，也可以将积分用于抵扣商品金额或其他特定用途。
余额交易：用户可以向系统充值余额，并在购物时使用余额支付。
购物车：用户可以将商品加入购物车，方便批量结算和管理购物清单。
商品管理：管理员可以添加、编辑、删除商品，并设置商品的属性和价格等信息。
